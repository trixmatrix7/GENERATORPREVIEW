# Preset-Format

Die Presets liegen unter `preset/*.chainwtf-preset.json` im STANDARDISIERTEN
`chainwtf-game-preset` v2 (euer Schema). Math ist inline (`math.mode: "inline"`,
`rtpBps` = operativer zertifizierter RTP), Sizing-Paket unter `extras.sizing`,
kompletter 1:1-Tuning-Block unter `extras.presentationTuning` (Vice).
Das alte v1 `vice-heat.preset.json` ist entfernt.
