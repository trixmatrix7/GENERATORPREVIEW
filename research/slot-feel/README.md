# Slot-Feel Database

The reference library for "das echte Slot-Gefühl" — how the top studios
(Pragmatic Play, Nolimit City, Hacksaw Gaming, Stake exclusives/Paperclip)
build presentation, pacing and ESPECIALLY sound so that everything harmonizes.
Built from a deep web-research sweep (2026-07-16); each report carries its
sources. Estimates are marked as estimates.

## How to use this

1. Start with [07-gap-analysis](07-gap-analysis-our-games.md) — the concrete,
   prioritized list of what OUR games (Vice Heat, Crack Farm) are missing,
   each mapped to the exact config/file to change.
2. [00-our-current-state](00-our-current-state.md) is the baseline inventory
   of every feel parameter we run today.
3. The studio reports (01–04) are the reference anatomy per house style.
4. [05-sound-design](05-sound-design.md) is the audio bible — read before
   authoring ANY new sound. [06-game-feel](06-game-feel-juice.md) is the
   timing/animation bible — read before tuning ANY animation.
