# Preset-Format

`preset/vice-heat.chainwtf-preset.json` ist das VICE-HEAT-Preset im
standardisierten `chainwtf-game-preset` v2 (euer Schema, 0 Validierungsfehler).
Math inline (`rtpBps` = operativer zertifizierter RTP, inkl. custom.viceBuyStages
+ custom.anteBet), Sizing-Paket unter `extras.sizing`, kompletter 1:1-Tuning-Block
unter `extras.presentationTuning`. Dieses Paket enthaelt NUR Vice Heat.
